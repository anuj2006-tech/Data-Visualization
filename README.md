# IPL Data Visualization Project 

##  Project Overview
This project performs exploratory data analysis (EDA) and visualization on IPL dataset using Python.

##  Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Key Analysis
- Team performance comparison
- Top run scorers
- Match win distribution
- Toss impact analysis
- Seasonal trends

##  Dataset
IPL historical match dataset.

##  Objective
To analyze IPL data and extract meaningful insights using data visualization techniques.
